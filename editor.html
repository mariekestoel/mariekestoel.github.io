Instellingen

Met een token kunt u wijzigingen direct opslaan op GitHub. Zonder token kunt u alleen de JSON-bestanden downloaden en daarna handmatig op GitHub uploaden.

Help: een nieuw token aanmaken

  1. Ga naar github.com/settings/personal-access-tokens/new en log in.
  2. Geef een naam, bv. mariekestoel-editor.
  3. Bij Expiration: kies “No expiration”.
  4. Bij Repository access: “Only select repositories” → kies mariekestoel.github.io.
  5. Bij Repository permissions: zet Contents op Read and write. Niets anders.
  6. Klik Generate token, kopieer de code, plak hierboven, klik Token bewaren.
