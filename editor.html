Bezig met laden…

Klik op een jaar om de concerten te zien en te bewerken. "Bewaren op GitHub" zet de wijzigingen direct online (token nodig). "Download agenda.json" maakt een bestand dat u handmatig naar GitHub kunt slepen.

Instellingen

Met een token kunt u wijzigingen direct opslaan op GitHub. Zonder token kunt u alleen agenda.json downloaden en daarna handmatig op GitHub uploaden. De token wordt alleen in deze browser bewaard, niet online.

Help: een nieuw token aanmaken

  1. Ga naar github.com/settings/personal-access-tokens/new en log in.
  2. Geef een naam, bv. mariekestoel-editor.
  3. Bij Expiration: kies "No expiration" (of een lange tijd).
  4. Bij Repository access: kies "Only select repositories" en selecteer deze website-repository.
  5. Bij Repository permissions: zoek Contents en zet die op Read and write. Niets anders.
  6. Klik onderaan Generate token en kopieer de lange code.
  7. Plak de code hierboven en klik Token bewaren.
